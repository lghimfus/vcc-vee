# Viagogo Coding Challenge

## How to run

Clone the repository.
```
git clone https://github.com/lghimfus/vcc-vee.git
```
Execute the following command from the root folder of the project.
```
./gradlew run -q
```

## Built with 
* [Gradle](https://gradle.org/)
